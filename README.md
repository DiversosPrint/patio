# Controle de Pátio Diversos Print

Sistema independente para localização e liberação de carretas e rodotrens.

## Executar

1. Abra o PowerShell nesta pasta.
2. Execute `npm start`.
3. Acesse `http://localhost:3010`.

Usuários: `admin` / `1234` e `coordenador` / `4321`. As senhas podem ser personalizadas pelas variáveis `PATIO_ADMIN_PASSWORD` e `PATIO_COORD_PASSWORD`.

## Hospedagem

O projeto inclui `render.yaml` para publicação como Web Service no Render. As credenciais online são configuradas pelas variáveis de ambiente indicadas acima.
